## 🚀 Project Overview

A robust **RESTful Backend-as-a-Service** designed to manage personal finances with precision. This API enables users to securely track daily expenses, set budget thresholds, and gain insights through automated monthly analytics.

Built with **Node.js** and **Express**, using **PostgreSQL** for relational data integrity, this project demonstrates high-performance API design and secure authentication practices.

### 🛠️ Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** PostgreSQL (with Sequelize/pg-pool)
* **Security:** JWT (JSON Web Tokens) & Bcrypt hashing
* **Environment:** Replit / GitHub

---

## ✨ Key Features & "Merit" Highlights

* **🔐 Secure Authentication:** Implemented custom JWT middleware to protect user-specific data and handle session-based authorization.
* **📈 Monthly Analytics:** Built a data aggregation engine that calculates total spending by category (Food, Rent, Travel, etc.) for the current month.
* **⚠️ Smart Budget Alerts:** Integrated logic that triggers a warning flag in the API response when a user exceeds their defined monthly budget.
* **📂 Structured Data:** Designed a relational schema to ensure 100% data consistency between Users and Expense records.


## ⚙️ Setup & Execution

1. **Clone & Install:**
```bash
git clone https://github.com/Aaryash08/Smart-Expense.git
npm install

```


2. **Environment Variables:**
Create a `.env` file with `DATABASE_URL` and `JWT_SECRET`.
3. **Run the Server:**
```bash
npm start

```
